Self Balancing Robot

Description

Developed this robot in e-Yantra Robotics Competition (eYRC 2019-20), theme “Biped Patrol”.
This is based on the classic inverted pendulum control problem. The robot attempts to balance from an
induced tilt angle by moving forward or backward, the same way we try to balance a stick on a finger.

Project start date : Oct 2018

Project end date : July 2019
